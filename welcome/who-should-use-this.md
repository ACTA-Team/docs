# Who Should Use This

Built for teams that need reliable, programmable trust in records.

Platforms using credentials

- Universities, certifiers, marketplaces, DAOs
- Value: reduce fraud, enable instant verification, keep apps stateless

Builders & product teams

- Solo devs, dApp prototypes, web3 platforms
- Value: plug‑and‑play issuance/verification; start on testnet, scale to mainnet

Enterprise use

- Projects replacing costly custodial validation
- Value: auditable on‑chain state and encrypted off‑chain data; low operational overhead
